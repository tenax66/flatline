# デザインガイド

19世紀博物学的な素材（図譜・標本・地図・青写真）と、モダンな形式（グリッド・番号・ラベル）の融合を目指す

## 1. 原則

1. **紙らしさ** 光沢・グロー・ガラス（blur）・グラデーション・ドロップシャドウを使わない。
2. **グリッドを見せる** 要素をグリッドに整列させる。グリッドは薄く（文字色の 4〜6% アルファ）見せる。
3. **図版（プレート）の構造を借りる** 区切りは細い罫（単線・二重線）。図にはキャプションを添える。枠線は 1px のヘアラインまでにする。太い枠・大きな角丸は使わない。
4. **二つの紙、二つの印画法** ライトテーマは「製図用紙にインク」、ダークテーマは「シアノタイプ（青写真）」をモチーフとする。
5. **顔料の色** 紙・インク・テラコッタ（土）・プルシアンブルー（鉱物）の色を基調とする。彩度の高い色は「標本の差し色」（朱・黄・緑）のように用いる。ネオン・パステル・純白 #fff・純黒 #000 は使わない。
6. **動きは静的に** アニメーションは紙のふるまい（フェード、ページ繰り）までで、パララックス・バウンス・回転は使わない。
7. **紙のテクスチャ** 紙ノイズ（feTurbulence 等）を全面に薄くかけることを推奨する。テクスチャの重ねがけはしない。
8. **ピクセルフォントの維持** 短歌本文のピクセルフォント（dotgothic16 / lores-12）と絵文字は常に維持する。

## 2. トークン

| トークン | ライト（製図用紙） | ダーク（青写真） |
|---|---|---|
| 紙 `--bs-body-bg` | `#f5e8d5` | `#11294d`（プルシアンブルー） |
| インク `--bs-body-color` | `#140d00` | `#f5e8d5` |
| リンク（地図のインク青） | `#2940b3` | `#9cc3ff` |
| アクセント `--fl-accent`（テラコッタ） | `#b0593a` | `#d28e6b` |
| 方眼グリッド | インク 5% アルファ / 28px | 同左（白線方眼になる） |
| ヘアライン罫 | インク 30〜60% アルファ / 1px | 同左 |
| 紙ノイズ | opacity 0.04 | 同左 |

- 角丸: 原則 0〜4px。ピル形・カード形の大きな角丸は使わない。
- 罫: 区切りの基本は二重ヘアライン（`hr`）。表組み・枠は単線ヘアライン。

## 3. コンポーネント指針

- **ナビバー**: 不透明な紙＋下辺ヘアライン。blur・半透明は使わない。
- **`hr`**: 図譜のプレート罫（1px 二重線、間隔 3px）。
- **作者名・日付**: インク 65%。
- **ページネーション**: 現在ページはテラコッタ。ホバーはテラコッタ 15% の面。
- **画像**: ヘアラインの枠＋下にキャプション（ラベル書式）を基本形とする。
- **強調**: 太字かテラコッタ。ハイライト（マーカー風背景）は使わない。

## 4. アンチパターン

- グラスモーフィズム、ニューモーフィズム、グロー、視差効果
- 大きな角丸、カードの影、色面で塗り分けた UI
- 純白・純黒、ネオン色、写真のフルブリード背景
- 装飾目的のストック画像
